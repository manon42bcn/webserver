Detailed Description

A simple logging utility for managing log messages with different levels of severity.

This class provides methods for logging messages with different severity levels, including DEBUG, INFO, WARNING, ERROR, and FATAL. It supports logging to the console or to a file, and allows for filtering messages based on a log level.

Constructs a Logger object with a specified log level and optional file logging.

This constructor initializes the logger with a specific log level, determining the minimum level of messages that will be logged. If file logging is enabled, the logger attempts to open a log file; if the file cannot be opened, logging defaults to the console.

Parameters
levelThe minimum log level for messages to be logged (e.g., LOG_DEBUG, LOG_INFO).
log_to_fileBoolean indicating whether to log messages to a file.
Note
If the provided log level is out of the allowed range (LOG_DEBUG to LOG_ERROR), the constructor will terminate the program.
