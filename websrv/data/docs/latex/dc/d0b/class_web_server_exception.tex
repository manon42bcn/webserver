\doxysubsection{Detailed Description}
Custom exception class for handling errors within the web server project. 

This exception class provides a standard way to handle errors by allowing custom messages. It inherits from {\ttfamily std\+::exception} and overrides the {\ttfamily \doxylink{class_web_server_exception_ac8a96b6d2eb4ef391ab045314815100f}{what()}} method to return a message.


\begin{DoxyItemize}
\item Can be initialized with either {\ttfamily std\+::string} or {\ttfamily const char\texorpdfstring{$\ast$}{*}} for message flexibility.
\item The destructor is {\ttfamily noexcept}, ensuring that it does not throw exceptions during destruction.
