\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Web\+Server\+Cache$<$ T $>$}
A simple, in-\/memory Least Recently Used (LRU) cache. 

The {\ttfamily \doxylink{class_web_server_cache}{Web\+Server\+Cache}} class is designed to manage a fixed-\/size cache for storing frequently accessed items in a web server context. It utilizes a Least Recently Used (LRU) eviction policy, where the least recently accessed entry is removed when the cache reaches its maximum capacity.
